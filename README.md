<h1>Sentiment Analysis using Machine Learning</h1>

<h2>1. Project Overview</h2>
<p>
This project focuses on building a machine learning model to classify text comments as Positive or Negative.
</p>

<h2>2. Objectives</h2>
<ul>
  <li>Clean and preprocess text data</li>
  <li>Convert text into numerical features using TF-IDF</li>
  <li>Train a machine learning model</li>
  <li>Predict sentiment from new comments</li>
</ul>

<h2>3. Dataset</h2>

<h3>Dataset Description</h3>
<p>
The dataset contains text comments with sentiment labels.
</p>
